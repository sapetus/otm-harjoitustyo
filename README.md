## Space Invaders
Klassinen Space Invaders -henkinen peli. 11x5 vihollista liikkuvat edes takaisin alustalla, joita pelaajan tulee ampua ja näin kerätä pisteitä. Pisteistä pidetään kirjaa erillisessä tietokannassa.
